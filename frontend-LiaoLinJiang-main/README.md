## 作业内容 
采用Bootstrap框架开发一个系统前端页面，包含首页、登陆、注册页面。
### 要求
1. 三个不同页面这件能够相互跳转；
2. 使用到Bootstrap框架进行开发；
3. 要求登陆、注册页面具备前端的表单提交校验；
4. 页面美观；
5. 提交系统说明文档，包含页面的运行截图和系统描述（在**操作.md**文件中编写），使用Markdown标记语言编写。
